Making a Post on Reddit

Now that we've learned all about our account, let's make our first post on Reddit using Snoocore and Reddit's API.

We will be using the code below to make a post on the subreddit /r/Bottest.

Let's take a closer look at the code. First, to make a post to a subreddit, we need to know which subreddit to post to, the title of our post, and the content of our post. We can make a self post, whose content is comprised of text, or a link post, which when clicked, redirects to another site. In this case, we will make a link post. We declare those three pieces of information as variables, SUBREDDIT, TITLE, and LINK to be passed to a function later.

The oauth portion of the code is simple and straightforward. It is nearly same as the code from earlier, when we requested information about our account. However, note that the scope is different this time. On the Reddit API page, you can find what scope is necessary for a request by looking at the green box next to the request. For example, next to the /api/submit request, one can see that 'submit' is necessary in the scope.

The documentation walks us step-by-step through what parameters we need and what values they should be. We put the string 'json' as the api_type since that is what the instructions tell us to do. Same with kind, sr, title, etc. The documentation also explains why we pass the boolean value true is given to resubmit. If it was false, we'd get an error if another person already submitted the same link, which is pretty likely! Isn't Reddit's API documentation great?
