Getting a Reddit API Key

To use the Reddit API, we must first make a Reddit account and get an API Key.

  1. First, let's make an account on Reddit.

    2. p

  2. Once you have your account, log in and go to your app preferences.

  3. Click the "Create an app" button, and fill out the form shown. Name your app as you wish, check the "script" button, write a short description, and use "http://www.example.com/unused/redirect/uri" as the redirect uri (we will not be using the uri). Finish by clicking the "create app" button.

    4. p

  4. And voila! You should have received an API Key, along with a nifty client ID for your app. Don't let your ID or secret fall into the hands of any nefarious individuals! Some thief might steal your precious bot!
