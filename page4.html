Method #2: Using a Wrapper

Wrappers allow for easy access to Reddit's API and make its use much simpler. A list of unofficial clients for Reddit's API can be found here. In this tutorial, we will be using Snoocore, a Node.js Wrapper.

  1. First, make sure you have Node.js. If you don't, you can download it here.
  2. Make a directory for your bot. We will call our project directory BotFolder.
  3. In your directory, install Snoocore by typing npm install snoocore in Terminal or Command Line.

  4. Make a file named "app.js" in your project directory.
  5. In the "app.js" file, copy and paste the following code. Put in a name for your userAgent, such as "FuturamaBot by /u/BeamMeUpBotty". Put in your client ID, client secret, Reddit username, and Reddit password. Then, save the file.

  6. Now, back in Terminal or Command Line, run app.js by typing node app.js.
